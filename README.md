# Pranesh Portfolio - Video Editor Portfolio Website

A modern, animated portfolio website built with Angular and TypeScript for Pranesh, a professional video editor, reels creator, and photo editor.

## Features

- Modern, animated UI with smooth CSS animations
- Fully responsive design
- Sections for hero, about, services, portfolio gallery, and contact
- Gradient backgrounds and hover effects
- Built with Angular 18+ and TypeScript

## Setup Instructions

### 1. Add Your Profile Photo

Copy your profile image to:
```
src/assets/images/pranesh.jpg
```

Or update the image path in `src/app/components/hero/hero.component.html` (line 14)

### 2. Run the Development Server

```bash
ng serve
```

Navigate to `http://localhost:4200/` in your browser.

### 3. Add Your Reels/Videos to Portfolio

To add your reels to the portfolio gallery:

1. Open `src/app/components/portfolio-gallery/portfolio-gallery.component.html`
2. Replace the placeholder items with your actual video content

### 4. Customize Contact Information

Update your contact details in:
- `src/app/components/contact/contact.component.html`
