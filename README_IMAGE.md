# Adding Your Profile Image

## Quick Steps:

1. **Save your profile photo** to this exact location:
   ```
   d:\Pranesh Portfolio\portfolio\src\assets\images\pranesh.jpg
   ```

2. **The image should be**:
   - Named exactly: `pranesh.jpg`
   - Format: JPG, JPEG, or PNG (will be renamed to .jpg)
   - Recommended size: 800x800px or larger
   - Square or portrait orientation works best

3. **After adding the image**:
   - The portfolio will automatically display your photo
   - Refresh the browser if it's already running

## Manual Method:

1. Right-click on the image you want to use
2. Copy it or save it
3. Navigate to: `d:\Pranesh Portfolio\portfolio\src\assets\images\`
4. Paste the image and rename it to `pranesh.jpg`
